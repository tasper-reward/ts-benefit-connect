## Get Started / Help

I use [nvs](https://github.com/jasongin/nvs) for Node version switching on Windows.  
To install us WinGet:  
```bash
winget install jasongin.nvs
```

Than install the current LTS of NodeJs by running:
```bash
nvs add lts
nvs use lts
```
